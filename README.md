# Py3Downloader

## Description

Ce script à pour but de créer les listes de librairies pour des projets afin de les importer dans un dépôt PyPi local dans des environnements air-gapped

## Utilisation

Ajouter des listes de librairies au format **name==version** dans des fichiers **projet_name.list** dans le répertoire ./list/ 

Le script va écrire le résultat dans ./data/ au format .whl
